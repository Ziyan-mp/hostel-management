# Hostel Management System

A Hostel Management System developed using Python and MySQL to automate hostel administration tasks such as student registration, room allocation, fee management, and record maintenance.

## Features
- Student registration and management
- Room allocation system
- Fee management
- Database connectivity using MySQL
- Add, update, delete, and search records

## Technologies Used
- Python
- MySQL
- MySQL Connector
- Tkinter / Command Line Interface

## Purpose
This project helps reduce manual work and improves efficiency in managing hostel operations through a simple and user-friendly system.
